# ChessTest

Ruby exam. 
We have here two directories :
- First directory contains a simple implementation of the test
- Second directory goes a little bit further with optimization and large arrays

## Prerequisites

- Ruby 3
- rSpec

## Usage

Install rSpec

    gem install rspec

Then run the rb file:

    ruby chess.rb
 
## Specs

To run specs

    rspec chess_spec.rb
